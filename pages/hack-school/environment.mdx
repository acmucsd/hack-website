# Setting Up Your Environment

This page will walk you through setting up your coding environment. Our general recommendation is to use Visual Studio Code, a lightweight, powerful, and customizable
text editor that simplifies your development experience. Here are the instructions to set it up:


1. Download Visual Studio Code for your machine [here](https://code.visualstudio.com/Download). 

Visual Studio Code is a powerful editor that keeps your development experience simple, but offers helpful features, such as a debugger. It has a 
very well maintained ecosystem, with no shortage of extensions, themes, and language-specific feature packs that can enhance your development experience.

2. Download Node.js for your machine [here](https://nodejs.org/en/download).

Our development is going to be done in the MERN stack, which means all code is written in JavaScript/TypeScript. Node.js is a JavaScript runtime environment
that allows developers to execute JavaScript code outside of the browser. Originally, JavaScript was meant to be a scripting language for the web, and was only supported when executed in a browser environment. You 
can read a bit about the history [here] (https://javascript.info/intro).

The best way to learn something is by actually building with it, but this JavaScript tutorial is a great piece of documentation and quick
JavaScript lessons: https://javascript.info.

3. Install Git and create a GitHub account
